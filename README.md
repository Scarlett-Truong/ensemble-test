# ensemble-test

## How to use

- Clone the repo with this command.
```bash
git clone https://github.com/Scarlett-Truong/ensemble-test.git
```

- Run the back-end
```bash
cd server
yarn
node server.js
```

- Run the front-end
```bash
cd client
yarn
gatsby develop
```
- The website is available at `http://localhost:8000/`
